# AI Projects Playground

A collection of AI projects for learning and experimentation. Each project has its own isolated uv environment and configuration.

## Getting Started
1. Install [uv](https://github.com/astral-sh/uv)
2. Navigate to `projects/` directory for detailed instructions
3. Copy `project-template` to create a new project
4. Run `uv init` to set up environment
5. Start building your AI project!

## Project Structure
See `projects/README.md` for detailed project organization and management instructions.
